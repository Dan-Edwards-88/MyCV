# Dan Edwards - CV

Personal CV/resume site built with plain HTML and CSS.

## View

Live at: `https://dan-edwards-88.github.io/MyCV/`

## Local development

Just open `index.html` in a browser. No build step required.

## Files

- `index.html` - CV content and structure
- `styles.css` - all styling
- `README.md` - this file

## Print / PDF

Press `Ctrl+P` (or `Cmd+P` on Mac) to print or save as PDF. Print styles are included.
